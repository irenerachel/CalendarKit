# 更新日志

## v1.0.0 (2026-01-29)

### 新功能
- 月历/周历生成，支持完整、公历、填写三种样式
- 农历、节气、节假日显示
- 莫兰迪主题配色，8种预设颜色 + 自定义颜色
- 图片上传，支持拖拽移动和滚轮缩放
- 背景图功能，支持 Alt+拖拽移动、Alt+滚轮缩放
- 多种预设比例（小红书、抖音、B站、微博等）
- 自定义画布尺寸，支持 px 和 cm 单位
- 16款精选中文字体
- 字体大小调整（50%-200%）
- 标题分隔线，支持粗细调整（1-4px）
- 设置模板保存/加载/删除
- 自定义颜色保存（最多6个）
- 撤回/重做功能
- 多格式导出（PNG/JPG/SVG）
- 多倍率导出（1x/2x/3x/4x）

### 移动端适配
- 响应式布局（平板、手机、小屏手机）
- 触摸手势支持（单指拖动、双指缩放）
- 触摸设备按钮尺寸优化

### UI/UX
- 自定义字体下拉框（主题化样式）
- 自定义设置模板下拉框
- 轮播提示（内容超出、快捷键、保存设置等）
- 字体推荐标签
- IP形象水印
- 关注提示 + 二维码弹窗
