# CalendarKit 日历小助手

一款精美的在线日历生成工具，支持月历/周历制作，莫兰迪配色，适合社交媒体配图。

## 在线预览

**[点击使用 CalendarKit](https://irenerachel.github.io/CalendarKit/)**

## 功能特性

- 📅 **月历/周历** - 支持完整、公历、填写三种样式
- 🎨 **主题配色** - 莫兰迪色系，支持自定义颜色
- 🖼️ **图片插入** - 拖拽上传，支持缩放移动
- 🌄 **背景图** - 支持背景图片，可调透明度
- 📐 **多种比例** - 预设小红书、抖音等常用尺寸
- 🔤 **多字体** - 内置16款精选中文字体
- 💾 **模板保存** - 保存设置，12张日历风格统一
- 📱 **移动端适配** - 支持触摸手势操作
- 📤 **多格式导出** - PNG/JPG/SVG，支持1-4倍分辨率

## 使用方法

1. 选择月历或周历
2. 设置年份、月份
3. 选择比例、布局、主题色
4. 上传图片（可选）
5. 调整字体和样式
6. 点击"导出图片"

## 技术栈

- 纯前端实现（HTML + CSS + JavaScript）
- html2canvas 导出图片
- lunar-javascript 农历计算

## 本地运行

直接用浏览器打开 `index.html` 即可使用。
